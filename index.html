About

I drive from Brighton MA, to Rochester NH and back three times a week. I was tired of looking up event schedules to figure out if I should take Route 1 to Soldiers Field Road, or stay on 95 and take 90 the rest of the way, so I made this. It's using the SeatGeek API to find out if there is an event going on at Fenway or the Garden, and then taking the popularity of the event and recomending a route home. It's basically guessing whether the Levrrett Connector is going to be a shit storm; flipping a coin is so Web 2.0.

This little bit of the web doesn't take into account any accidents or construction, in Boston there will be both either way. I check this before I leave, and then keep my eyes on the road. Real-time updates wouldn't do any good while I'm commuting.
